# План разработки приложения "Карточки напоминаний"

## Общая информация о проекте

**Цель**: Создание кроссплатформенного приложения для управления карточками-напоминаниями

**Платформы**:

- Web (React)
- Mobile (React Native)
- Backend (PHP)

**Основной функционал**:

- Просмотр ленты карточек
- Авторизация пользователей
- Управление личной коллекцией
- Редактор карточек (CRUD операции)

### Задача 2.4: API для работы с карточками (3 дня)

**Описание**: CRUD операции для карточек

- Модель Card с валидацией
- GET /api/cards - получение списка карточек (с пагинацией)
    - Фильтрация по пользователю
    - Сортировка (по дате создания, названию)
- GET /api/cards/{id} - получение одной карточки
- POST /api/cards - создание карточки
    - Валидация полей (title, description обязательны)
    - Сохранение в БД
- PUT /api/cards/{id} - обновление карточки
- DELETE /api/cards/{id} - удаление карточки
- Проверка прав доступа (пользователь может редактировать только свои карточки)

**Результат**: Полный набор API для управления карточками
